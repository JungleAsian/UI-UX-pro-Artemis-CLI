# AI assistant usage

After installation, ask your coding assistant to use the Artemis UI/UX rules before generating frontend code.

## Recommended prompt

```text
Use the UI/UX Artemis guidance.

Product:
Audience:
Platform:
Brand tone:
Primary user journey:
Screens/components:
Tech stack:
Accessibility requirements:
```

## Best workflow

1. Generate design principles.
2. Generate design tokens.
3. Define the component system.
4. Create wireframe-level structure.
5. Implement frontend code.
6. Review the result for hierarchy, accessibility, and responsiveness.

## What Artemis should help with

- Design-system planning.
- UI critique.
- Component consistency.
- Layout and information hierarchy.
- Accessibility reminders.
- Frontend implementation constraints.

## What Artemis should not do

- Copy another product's proprietary UI.
- Add analytics, tracking, or remote dependencies without explicit approval.
- Prioritize decoration over task completion.
